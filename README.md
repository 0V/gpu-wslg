# gpu-wslg

This is a minimal template for using WSLg (Windows Subsystem for Linux GUI) with **NVIDIA GPU** on Docker Container (Ubuntu 20.04).  

## Require

- Windows 11
- WSL2 (latest)
- Docker Desktop 
  - docker
  - docker-compose V2
- GPU 

## Installation

```
docker compose up -d --build
docker compose exec gpu-wslg /bin/bash
```

## Try!
